# Title : 📊 Optimizing IT Support Team Performance Using Analytics

## 📌 Project Overview
This project focuses on analyzing IT support team data to identify performance gaps, improve operational efficiency, and optimize ticket resolution processes using data visualization and analytics.

The project was developed as part of my internship under Infosys Springboard.

---

## ❓ Problem Statement

IT support teams often handle large volumes of service tickets daily. However, organizations face challenges such as:

- Delayed ticket resolution
- SLA violations
- Uneven workload distribution
- Lack of performance visibility
- Poor data quality affecting decision-making

The goal of this project is to use analytics and data visualization techniques to identify inefficiencies and provide data-driven recommendations to improve IT support team performance.

---

## 📂 Dataset Description

The dataset used in this project contains IT support ticket information, including:

- Ticket ID
- Issue Category
- Priority Level (Low, Medium, High)
- Ticket Status (Open, Closed, In Progress)
- Assigned Technician
- Created Date
- Resolved Date
- Resolution Time
- SLA Status
- Customer Feedback Score

### 🧹 Data Preprocessing Steps:
- Removed duplicate records
- Handled missing values
- Replaced inconsistent category names
- Trimmed extra spaces
- Converted text case (lowercase/uppercase formatting)
- Corrected data types (Date, Numeric, Text)
- Standardized priority and status fields
